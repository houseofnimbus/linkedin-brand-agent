How this widget works

Every morning, the linkedin-daily-digest scheduled task fills this widget with the day's comment drafts. Click Approve, Edit, or Skip on each card. When you hit Submit & post, the agent posts your approved comments and likes each commented post. If nothing's here yet, the agent hasn't run its first pass — check the scheduled task status in Cowork, or trigger it manually with "run my LinkedIn agent".
