description: Run the LinkedIn Brand Agent setup wizard
---

Run the `linkedin-brand-agent-setup` skill. Walk the user through the 8 configuration
questions (brand positioning, allowed facts, target accounts, hashtags, topic-phrase
search terms, domain guardrails, competitor blocklist, engagement schedule), then
write their answers into their workspace, register the two scheduled tasks, and
create the LinkedIn Digest artifact.

If the user has already run setup before, confirm they want to re-run (re-running is
safe but will overwrite their config files).
